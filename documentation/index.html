Installing Template

After unzip the download pack, you'll found a Template Folder ( Fury ) with all the files. You can view this Template in any browser, you can display or edit the Template without an internet connection.

Now open your FTP Client (like Filezilla) and upload the content of the Template on your server root. Once the files are done uploading go to www.yourdomainname.com/index.html
